# Carcassonne
to run
python3 main.py
-игра Carcasson
-python pyQt6


Игрок выставляет фишки на игровом поле так, чтобы они составляли цельную картину. Рекку можно "подсоединить" только к реке, дорогу к дороге, город к городу, поле к полю.
